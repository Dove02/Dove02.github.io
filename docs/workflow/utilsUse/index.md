# 常用库使用和配置

## pdf.js

场景：Vue 组件中实现 PDF 的预览功能。用户无需下载，在线预览 PDF 文档。

环境：Vue 3 + pdfjs-dist 4.8.69

[点击查看](/workflow/utilsUse/pdfjs)


## jsdiff

场景：文本替换后，计算两段文本之间修改了多少字

依赖：`"diff": "^8.0.2"`

[点击查看](/workflow/utilsUse/jsdiff)
